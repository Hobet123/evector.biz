HTML Files and Structure

About Cadem~ Template

Cadem is a responsible HTML template with 12-column grid based on Twitter Bootstrap framework. This template has been designed as a pure corporate graphics solution suitable for business, non-profit organizations, website portfolio or product single sites with the support of responsive layout. This template uses the latest HTML5 and CSS3 technologies.

Columns

Template is divided into rows and columns, with respective classes .row and .span*, where rows divide horizontal parts of the page and columns divide each row into smaller vertical sections. Since this template is based on 12-column grid system, you can divide each row into various number of vertical sections as long as they sum up at twelve (12). Look at the examples below for better explanation and some real examples.

Full-width: To achieve a full-width column, use the code below:

Container

There's one more special class which is being used across all HTML files, which is .container. This class determines the maximum width of a page and also makes sure that page is center-aligned. It also determines rendering of a template on mobile devices and tablets.

CSS Files and Structure

This template comes with 6 CSS files:

css/bootstrap.min.js
css/bootstrap-responsive.min.js
css/bootstrap-override.css
css/styles.css
css/flexslider.css
css/font-awesome/font-awesome.css (plus css/font-awesome/font-awesome-ie7.css for IE compatibility)

Please note that the first two files are being loaded with css @import rule at the top of css/styles.css stylesheet.

About each file:

css/bootstrap.min.js
Twitter bootstrap framework. A base twitter bootstrap CSS files, which defines various elements like columns, navigation, tabs, icons, forms and buttons.

css/bootstrap-responsive.min.js
Twitter bootstrap support for responsive layout, so the template can be displayed on mobile devices or tablets.

css/bootstrap-override.css
This file is not an official part of twitter bootstrap, it contains few overrides of media queries of original theme.

css/flexslider.css
Flex Slider stylesheet file.

css/font-awesome/font-awesome.css (+ font-files)
This file includes custom font which is being used for all icons which you can see in this template. More information about Font Awesome 100% scalable icon pack can be found at http://fortawesome.github.com/Font-Awesome.

css/styles.css
Main Cadem~ template stylesheet file (uncompressed version). To make any changes to this template, you should only need to modify this file. The file is divided info different sections, while every section take name from a subpage of template features.

JavaScript Plugins and Scripts

This template comes with 5 Javascript files:

jquery-1.8.3.min
jQuery is a Javascript library that greatly reduces the amount of code that you must write.

bootstrap.min
Twitter bootstrap base JavaScript file, which allows us to use many javascript components like tabs, accordion menus, dropdowns, tooltips and popovers, alert messages and so on. More information and examples can be found at http://twitter.github.com/bootstrap/javascript.html

jquery.isotope.min.js
Used to filter portfolio items.

jquery.flexslider.js
FlexSlider 2 JavaScript file.

functions.js
The most important file for Cadem~ template, which defines some basic javascript functions.

This file is divided into different sections, while every section is responsible for different functionality.

Responsive Navigation
Creates a collapsible navigation for mobile devices.

Portfolio
Basic functionality for filtering portfolio items.

Newsletter Form
Provides functionality for newsletter form (works through ajax and included .php script)

Flex Initialize
Initialize Flex Sliders.

config.php
